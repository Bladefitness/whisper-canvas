# Getting Started

Set up everything you need to view and edit the training canvases in under 5 minutes.

---

## Step 1: Install an Editor

You need a code editor that supports the Excalidraw extension. Pick one:

| Editor | Download | Notes |
|--------|----------|-------|
| **Cursor** (recommended) | [cursor.sh](https://cursor.sh/) | AI-powered editor, built on VS Code |
| **VS Code** | [code.visualstudio.com](https://code.visualstudio.com/) | The original, widely used |

Both are free. If you already have either installed, skip to Step 2.

## Step 2: Install the Excalidraw Extension

1. Open your editor
2. Click the **Extensions** icon in the left sidebar (or press `Cmd+Shift+X` / `Ctrl+Shift+X`)
3. Search for **"Excalidraw"**
4. Install **Excalidraw Editor** by `pomdtr` (extension ID: `pomdtr.excalidraw-editor`)
5. Reload the editor if prompted

That's it. Any `.excalidraw` file will now open as an interactive canvas.

## Step 3: Clone the Repository

Open your terminal and run:

```bash
git clone https://github.com/emekaajufo/clinic-growth-toolkit.git
```

Or download as a ZIP from GitHub (click the green "Code" button > "Download ZIP").

## Step 4: Open a Canvas

1. Open the `clinic-growth-toolkit` folder in your editor (`File > Open Folder`)
2. Navigate to any training folder (e.g., `trainings/01-plug-your-bottlenecks/`)
3. Click `canvas.excalidraw`
4. The canvas opens with all embedded images and text

### Navigation

- **Zoom:** Scroll wheel or pinch on trackpad
- **Pan:** Click and drag on empty space (or hold Space + drag)
- **Select:** Click on any element
- **Zoom to fit:** Press `Cmd+Shift+1` / `Ctrl+Shift+1`

## Step 5: Read the Training Script

Each training folder also has a `training.md` file. This is the full written training — you can:

- Read it directly in your editor (it renders as formatted markdown)
- View it on GitHub (markdown renders automatically)
- Use it as a script for recording video content

---

## Alternative: View Without Installing Anything

If you don't want to install an editor:

1. Go to [excalidraw.com](https://excalidraw.com)
2. Drag any `.excalidraw` file from your computer onto the browser window
3. The canvas loads with all images and text

Note: This is view-only for files with embedded images. For full editing, use the extension.
