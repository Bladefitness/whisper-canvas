# Style B Enhanced — AI Image Generation Prompts

Copy-paste these prompts to generate consistent isometric 3D glass morphism infographics.

---

## Color Palette

| Color | Hex | Role |
|-------|-----|------|
| Blue | `#4A7BF7` | Primary, trust, authority |
| Purple | `#7C5CFC` | Premium, special, processing |
| Gold | `#D4A843` | Value, achievement, highlight |
| Red | `#E63946` | Urgency, critical, clinical |
| White | `#FFFFFF` | Background (always pure white) |

---

## Base Style Block

Append this to every prompt:

```
Style requirements:
- ISOMETRIC 3D perspective with depth
- Frosted glass / glass morphism cards
- Soft gradients, subtle drop shadows
- Color palette: blue #4A7BF7, purple #7C5CFC, gold #D4A843, red #E63946
- PURE WHITE background (#FFFFFF)
- Clean sans-serif typography
- 4:3 landscape format
- Professional, premium feel
- NO text smaller than 14pt
- Visual hierarchy with size and color contrast
```

```

1. [STAGE 1 NAME] - [brief description]
2. [STAGE 2 NAME] - [brief description]
3. [STAGE 3 NAME] - [brief description]

Each stage should be a floating glass card with an icon and label.
Connect stages with glowing arrows.
```


### Equation / Formula

```

Create an ISOMETRIC 3D glass morphism infographic showing an audit checklist.
```

### Clinical Sidebar (Medical Analogy)

```
Create an ISOMETRIC 3D glass morphism infographic showing a clinical medicine concept applied to marketing.

Use frosted glass cards with soft gradients floating on layers.

Clinical concept: [CONCEPT NAME]
Marketing application: [HOW IT APPLIES]

Show the medical concept on the left (with medical icons like stethoscope, chart, pills)
and the marketing parallel on the right (with marketing icons like megaphone, funnel, chart).

Connect them with a "=" or "↔" symbol.

Use RED #E63946 as the primary accent color (clinical theme).

Style requirements:
- ISOMETRIC 3D perspective with depth
- Frosted glass / glass morphism cards
- Soft gradients, subtle drop shadows
- Primary color: red #E63946 (clinical accent)
- Secondary colors: blue #4A7BF7, purple #7C5CFC
- PURE WHITE background (#FFFFFF)
- Clean sans-serif typography
- 4:3 landscape format
```

## Tips

1. **Be specific** — "Show 7 stages" is better than "show a process"
2. **Name the colors** — "blue card" tells the AI which color to use where
3. **Include data** — Real numbers and percentages make visuals more compelling
4. **Generate 2-3 versions** — Pick the best one from each batch
5. **4:3 landscape** — This ratio fits perfectly in the Excalidraw canvas layout (620x465px image area)
