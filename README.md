# The Clinic Owner's Growth System

**5 free trainings that'll transform how you think about marketing your practice.**

Built with [Excalidraw](https://excalidraw.com) visual canvases and AI-generated infographics. Open any `.excalidraw` file to get an interactive, zoomable visual companion for each training.

## What's Inside

| # | Training | What You'll Learn |
|---|----------|-------------------|
| 1 | [Plug Your Bottlenecks](trainings/01-plug-your-bottlenecks/) | The 7-stage system: reputation, content, lead magnets, speed to lead, automation, 24/7 availability, upsell & bundle |
| 2 | [The 3-5% Rule (TOFU/MOFU/BOFU)](trainings/02-tofu-mofu-bofu/) | Why you're marketing to the wrong people and the full-funnel fix |

Each training includes:
- A **full training script** (markdown) you can use as a video script, workshop outline, or study guide
- An **Excalidraw canvas** with embedded AI-generated infographics and key takeaway text
- **Clinical medicine analogies** (NNT, PPV, dose-response, triage) that connect marketing concepts to clinical thinking

## Quick Start

### 1. Clone the repo

```bash
git clone https://github.com/emekaajufo/clinic-growth-toolkit.git
cd clinic-growth-toolkit
```

### 2. Install the Excalidraw extension

Open [VS Code](https://code.visualstudio.com/) or [Cursor](https://cursor.sh/) and install:

**Extension:** `pomdtr.excalidraw-editor`

Search for "Excalidraw" in the Extensions marketplace, or run:
```
ext install pomdtr.excalidraw-editor
```

### 3. Open any canvas

Navigate to any training folder and click the `.excalidraw` file. It opens directly in the visual editor. Zoom in, pan around, and explore each frame.

> You can also drag any `.excalidraw` file to [excalidraw.com](https://excalidraw.com) to view it in the browser.

## How It Was Made

Every visual in this toolkit was created using two tools:

1. **AI Image Generation** (Google Gemini) — Isometric 3D glass morphism infographics generated with a consistent visual style. See [guides/generating-visuals.md](guides/generating-visuals.md) for the exact prompts and workflow.

2. **Excalidraw** — The images are embedded into Excalidraw canvases with native text boxes for key takeaways, creating interactive visual companions you can zoom, edit, and export.

Want to create your own? Check the [guides](guides/) folder.
